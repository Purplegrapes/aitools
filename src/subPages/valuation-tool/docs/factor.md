# 一、指标说明文案

## 1. 近一年表现

### 指标名称

**近一年表现**

### 面向用户的简短说明

**看这只基金过去一年整体表现怎么样。**

### 白话解释

它反映这只基金在过去一年里是涨了还是跌了、幅度大概有多大。
这个指标能帮你快速判断：

* 它最近这一年的表现强不强
* 它是不是处在比较强势的阶段
* 但它只代表过去一年，不代表未来一定继续这样

### 推荐前台短说明

**过去一年涨得怎么样**

### 可放在 Tooltip / 说明弹层里的版本

近一年表现是指这只基金过去 1 年的累计涨跌幅。
它能帮助你快速了解这只基金最近一年的整体表现，但不能单独代表未来收益，也要结合风险和回撤一起看。

---

## 2. 历史最大跌幅

### 指标名称

**历史最大跌幅**

### 面向用户的简短说明

**看这只基金最差的时候，大概跌过多少。**

### 白话解释

它反映的是：
如果你在过去一个高点买入，最差的时候最多可能经历多大的亏损。

这个指标能帮你快速判断：

* 这只基金跌起来会不会很痛
* 它的抗跌能力大概怎么样
* 它适不适合风险承受能力较低的人

### 推荐前台短说明

**最差时大概跌过多少**

### 可放在 Tooltip / 说明弹层里的版本

历史最大跌幅是这只基金在一段历史时间里，从最高点跌到最低点的最大跌幅。
它能帮助你理解这只基金过去最极端的回撤有多大，也能帮助判断它的波动风险。

---

## 3. 费率

### 指标名称

**费率**

### 面向用户的简短说明

**看长期持有这只基金的成本高不高。**

### 白话解释

费率代表你持有这只基金需要支付的管理和运作成本。
虽然每天感觉不明显，但长期持有时，费率越高，对收益的影响越大。

这个指标能帮你快速判断：

* 这只基金长期拿着贵不贵
* 同类基金里它的成本是不是偏高
* 是否适合长期配置

### 推荐前台短说明

**长期持有成本高不高**

### 可放在 Tooltip / 说明弹层里的版本

费率是持有基金时需要承担的年度费用比例。
费率越低，长期持有时越有利；费率越高，长期收益越容易被成本侵蚀。
在同类基金中，费率通常是一个很重要的比较指标。

# 二、指标值评价映射关系

这里给你一版 **前台产品可直接使用的映射规则**。
注意：这套规则是 **面向普通用户的产品化规则**，不是严格金融行业标准，重点是帮助用户快速判断。

---

## 1. 近一年表现映射

### 评价逻辑

近一年表现越高，不代表一定越好，但能反映最近一年的强弱状态。

| 近一年表现     | 评价标签 | 前台解释       |
| --------- | ---- | ---------- |
| ≥ 20%     | 较强   | 过去一年整体表现较强 |
| 10% ~ 20% | 不错   | 过去一年表现不错   |
| 0% ~ 10%  | 一般   | 过去一年表现一般   |
| -10% ~ 0% | 偏弱   | 过去一年表现偏弱   |
| < -10%    | 较弱   | 过去一年整体偏弱   |

### 建议前台输出格式

#### 示例 1

**近一年表现**
+18.4%
**过去一年表现不错**

#### 示例 2

**近一年表现**
-12.6%
**过去一年整体偏弱**

### 使用提醒

* 行业基金、主题基金波动大，可以适当放宽标准
* 宽基基金、红利基金可以更偏保守判断

---

## 2. 历史最大跌幅映射

### 评价逻辑

绝对值越大，说明历史上最极端的亏损越深，风险感受通常越强。

| 历史最大跌幅      | 评价标签 | 前台解释         |
| ----------- | ---- | ------------ |
| 0% ~ -10%   | 较稳   | 历史回撤相对较小     |
| -10% ~ -20% | 稳健   | 回撤可控，整体相对稳健  |
| -20% ~ -30% | 中等   | 历史回撤中等       |
| -30% ~ -40% | 偏大   | 历史回撤偏大       |
| < -40%      | 较大   | 历史上跌幅较深，波动较大 |

### 建议前台输出格式

#### 示例 1

**历史最大跌幅**
-18.2%
**回撤可控，整体相对稳健**

#### 示例 2

**历史最大跌幅**
-36.5%
**历史回撤偏大**

### 更白话的替代文案

你也可以把“评价”写得更口语一点：

| 历史最大跌幅      | 更白话版本       |
| ----------- | ----------- |
| 0% ~ -10%   | 最差时跌幅不大     |
| -10% ~ -20% | 最差时跌幅还算可控   |
| -20% ~ -30% | 最差时大概跌过 2 成 |
| -30% ~ -40% | 最差时大概跌过 3 成 |
| < -40%      | 最差时跌得比较深    |

---

## 3. 费率映射

### 评价逻辑

费率越低，越适合长期持有。
这里建议做成 **长期成本评价**。

| 年费率           | 评价标签 | 前台解释     |
| ------------- | ---- | -------- |
| ≤ 0.30%       | 较低   | 长期持有成本较低 |
| 0.30% ~ 0.60% | 中等   | 长期持有成本中等 |
| 0.60% ~ 1.00% | 偏高   | 长期持有成本偏高 |
| > 1.00%       | 较高   | 长期持有成本较高 |

### 建议前台输出格式

#### 示例 1

**费率**
0.50% / 年
**长期持有成本中等**

#### 示例 2

**费率**
1.20% / 年
**长期持有成本较高**

### 使用提醒

* 不同基金类型费率天然不同
* 货币、债券、指数、主动权益不能完全一刀切
* 如果你未来做“同类对比”，建议再叠加一个“同类中偏低/偏高”的评价
