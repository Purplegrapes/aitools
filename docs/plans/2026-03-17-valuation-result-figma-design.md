# 基金结果页 Figma 对齐设计

## 背景

- 用户要求基于 Figma 节点 `35:263` 调整基金结果页视觉
- 当前产品决策已确认保留新指标：最大回撤、夏普比率、卡玛比率
- 当前产品决策已确认删除“谁更适合关注”

## 设计结论

- 只复用 Figma 的版式和视觉层级，不回退旧文案和旧指标
- `result.vue` 继续作为组合层，不改数据请求和状态控制
- 视觉调整集中在 `DetailSummaryCards.vue`

## 组件边界

- `result.vue`
  - 负责页面状态切换、接口结果整合、底部操作栏
- `DetailSummaryCards.vue`
  - 负责“今日参考”和“快速看懂这只基金”的视觉呈现
  - 保持 props 输入，不新增跨组件状态

## 实施要点

- 三条指标改成更贴近 Figma 的浅底行卡片，右侧数值与评价上下排列
- 指标名称后保留 tip 图标，点击底部弹层解释指标含义
- 定义说明改成普通段落文本，不再放在强调卡片中
- “主要跟踪 / 覆盖方向”改成 Figma 风格的并排浅色信息卡
- “日常波动预感”改成标题 + 等级标签 + 说明文案的结构

## 风险与验证

- 不新增接口字段，风险主要在视觉回归
- 需要确认底部弹层仍不会被操作栏遮挡
- 需要确认 OTC 与场内基金两种顶部行情展示没有被样式改坏
